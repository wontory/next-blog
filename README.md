## 컨벤션

- feat: 새로운 기능 추가
- fix: 오류 수정
- test: 테스트 코드 작성
- design: UI 디자인 변경
- style: 코드 포맷 변경
- refactor: 코드 리팩터링
- docs: 문서 작업
- chore: 패키지 매니저, 빌드 등
